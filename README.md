# lab08-NASA
This is only part of an assignment, the objective here isn't to create a functional application or code.

## Project Idea
Create a TODO List application, to help the NASA engineers to keep track of their objectives:
- Tasks can have different priority levels (lowest, low, medium, high, highest).
- Tasks can be assigned to different people, however if unassigned, anyone can "pick up" the task.
- Tasks can be marked as complete, deleted, or created.
- A logs tab, for admins only, will log every tasks created, deleted, marked as compelete, or picked up.
